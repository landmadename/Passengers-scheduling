# 乘客调度系统

一个调度系统
给出乘客的坐标，司机的坐标和座位数
返回乘客与司机的绑定关系


### 输入

- 订单id，用户坐标
- 司机id，司机坐标
- 模式：接 还是 送
- debug ：True or False

### 输出

- 订单id，司机id   绑定好的表



- 0.1 K-means 分配乘客
- 0.2 根据总路程优化
- 0.3 根据总路程和均方差优化
- 0.4 处理乘客太远的情况
- 0.5 考虑接和送乘客两种情况
